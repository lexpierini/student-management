# STUDENT MANAGEMENT

Projet d'étude centré sur l'authentification JWT, .Net Core et Entity Framework Core. CRUD simple pour le contrôle des étudiants.

## Technologies utilisées
- Asp.Net Core;
- Entity Framework Core;
- JWT Authentication;
- Swagger;
- PostgreSQL;
- React;
- Redux;
- Redux Toolkit;
- React Router v6;
- Axios;
- MUIDataTable;
- MUI v5.4.3;
